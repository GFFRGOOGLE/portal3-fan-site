📥 УСТАНОВКА

Главная Установка Моды для Minecraft Чит-коды Поддержка

🖥️ Windows

  • Скачайте NW.js SDK для Windows: nwjs.io
  • Создайте папку game и положите в неё файл index.html (код игры есть на сайте).
  • В той же папке создайте package.json (содержимое — ниже).
  • Перетащите папку game на nw.exe — игра запустится.
  • Готовый EXE: можете собрать сами за 2 минуты.

📱 Android

  • Способ 1 (простой): откройте страницу с игрой в Chrome на телефоне → нажмите «Добавить на главный экран» → игра станет приложением.
  • Способ 2 (APK): загрузите index.html на любой хостинг, затем на pwa2apk.com вставьте ссылку и скачайте APK.
  • Способ 3 (для разработчиков): используйте Android Studio с WebView.

🍏 macOS

  • Скачайте NW.js для macOS (файл .dmg) с nwjs.io.
  • Поместите папку game с index.html и package.json рядом с nwjs.app.
  • Запустите nwjs.app — игра откроется.
  • Для создания .app используйте Electron (инструкция в разделе поддержки).

📄 Пример package.json

{
  "name": "portal3-fan",
  "main": "index.html",
  "window": {
    "title": "Portal 3: Fan Edition",
    "width": 1024,
    "height": 768,
    "resizable": true,
    "fullscreen": false
  }
}
