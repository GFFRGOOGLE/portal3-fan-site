📥 УСТАНОВКА

Главная Установка Моды для Minecraft Чит-коды Поддержка

🖥️ Windows

  1. Скачайте NW.js SDK для Windows с официального сайта: nwjs.io.
  2. Распакуйте архив в любую папку (например, C:\nwjs).
  3. Создайте папку game внутри nwjs.
  4. Положите в неё файл play.html (код игры есть на сайте) и файл package.json (пример ниже).
  5. Перетащите папку game на файл nw.exe — игра запустится.
  6. Для удобства можете создать ярлык на рабочем столе.

Готовый EXE: вы можете собрать самостоятельно по этой инструкции или воспользоваться онлайн-конвертерами (например, PWA2APK для APK).

📱 Android

  1. Способ 1 (простой): откройте страницу с игрой в браузере Chrome на телефоне → нажмите «Добавить на главный экран» → игра станет отдельным приложением.
  2. Способ 2 (APK): загрузите play.html на любой хостинг (например, GitHub Pages), затем перейдите на pwa2apk.com, вставьте ссылку на ваш play.html и скачайте готовый APK.
  3. Способ 3 (для разработчиков): используйте Android Studio с WebView (более сложный, требует установки Android Studio).

Примечание: при установке APK разрешите установку из неизвестных источников.

🍏 macOS

  1. Скачайте NW.js для macOS (файл .dmg) с nwjs.io.
  2. Откройте .dmg и перетащите nwjs.app в папку Applications.
  3. Создайте папку game рядом с nwjs.app (или внутри неё) и положите туда play.html и package.json.
  4. Запустите nwjs.app — игра откроется.
  5. Если система не даёт запустить, разрешите в настройках безопасности.

Готовый .app: вы также можете упаковать игру в отдельное приложение с помощью Electron (инструкция в разделе поддержки).

📄 Пример файла package.json

{
  "name": "portal3-fan",
  "main": "play.html",
  "window": {
    "title": "Portal 3: Fan Edition",
    "width": 1024,
    "height": 768,
    "resizable": true,
    "fullscreen": false
  }
}

Сохраните этот код как package.json в папке game.
